# 全新设计方向探索（3 套）

> 按 `huashu-design` 设计方向顾问方法，用**三套互补逻辑**各推一个方向，覆盖从「东方文人」到「极客精确」到「暖光亲和」的光谱。三套都是为「子不语」这个**文气的技术个人博客 + AI 英语子应用**量身长出来的，**均完整保留昼/夜切换 + 首页配图切换**。
>
> 本轮以文档表达（色板/排版/气质/参考/布局示意）。**你选定一套或混搭后，我再用 HTML 做该方向的高保真原型（1–2 个关键页），眼见为实后再定稿、补全 [03-design-spec.md](./03-design-spec.md) 的具体色值。**
>
> 命名缘起：「子不语」典出《论语》"子不语怪力乱神"——克制、理性、有书卷气。这成为三个方向的共同精神底色，只是表达方式不同。

## 方向 A · 子曰·墨（Quiet Ink）

**一句话**：东方文人编辑风——暖纸 + 浓墨 + 一点朱砂，大留白，宋体标题，像一本安静的线装杂志。

- **设计逻辑**：🧠 最佳设计师视角（原研哉「白」的留白哲学 + Kinfolk 编辑排版）。
- **气质关键词**：克制、书卷、留白、温度、慢。
- **为什么适合**：最贴合「子不语」的文学气质；个人博客的核心是「读」，编辑风把阅读体验做到极致；与众多 Element 默认博客拉开差距。

| 品牌主色（朱砂） | `#C8453B` | `#E06A5E` |

| 次级区块 | `#F4EEE2` | `#1F1B16` |
| 卡片 | `#FFFDF8` | `#211D17` |
| 正文 | `#2B2620` | `#E9E2D4` |
| 次要文字 | `#6B6356` | `#A89E8C` |
| 边框 | `#E3DACA` | `#332C22` |

> 语义色沿用规范默认（success 墨绿 / warning 赭黄 / danger 朱砂同源 / info 黛蓝），整体收敛在暖色温层。

### 排版
- **标题**：思源宋体 / Noto Serif SC（display），有笔锋、有文气。
- **正文**：思源黑体 / Noto Sans SC，18px / 行高 1.8 / `max-width: 70ch`。
- 大量留白（页面留白 ≥ 50%），章节用细线 + 朱砂小点分隔。

### 签名细节
- 朱砂作「印章/标记」语言：作者印、置顶标记、链接 hover 下划线。
- 文章首字下沉（drop cap）可选。
- 昼夜 Hero 图：白天用暖调实景/水墨，黑夜用同构夜景，朱砂始终是唯一高饱和点。

## 方向 B · 终端·工坊（Terminal Workshop）

**一句话**：极客精确风——中性灰 + 单一电光蓝 + 等宽点缀，强网格、强对比，像 Vercel / Linear 的开发者站。

- **设计逻辑**：🏆 现实参照（Vercel / Linear / GitHub 的设计语言迁移）。
- **气质关键词**：精确、冷静、信息密度、工程感、现代。
- **为什么适合**：如果你希望博客突出「技术」属性、AI 英语子应用作为「工具」无缝融入，这套最自洽；网格与令牌天然适配本次系统化改造。
- **反 slop 注意**：刻意避开「均匀深蓝底 + 通用霓虹 glow」那种偷懒暗色——这里用**中性灰阶 + 单一克制电光蓝**，靠排版精度而非发光效果取胜。

### 色板

| 角色 | 白天 | 黑夜 |
|------|------|------|
| 品牌主色（电光蓝） | `#2563EB` | `#5B8DEF` |
| 页面底 | `#FFFFFF` | `#0A0A0A` |
| 次级区块 | `#FAFAFA` | `#141414` |
| 卡片 | `#FFFFFF` | `#161616` |
| 正文 | `#171717` | `#EDEDED` |
| 次要文字 | `#525252` | `#A1A1A1` |
| 边框 | `#E5E5E5` | `#262626` |

> 灰阶用 Neutral 系（接近 Tailwind neutral），主色是唯一彩色锚点。

### 排版
- **标题**：Inter Tight / Space Grotesk（几何无衬线，紧凑有力）。
- **正文**：Inter + Noto Sans SC，16–18px / 行高 1.7。
- **等宽点缀**：标签、日期、代码、数据用 mono（`SF Mono`），强化工程感。

### 签名细节
- 1px 精确分隔线 + hairline 边框；按钮/卡片直角偏小圆角（`--radius-md`）。
- hover 用主色 underline / border，不用阴影堆叠；focus ring 明确。
- 昼夜 Hero 图：白天浅网格底 + 实景，黑夜纯黑底 + 同图，主色点缀一致。

│  │ 摘要 灰     │ │ 摘要 灰     │ │ 摘要 灰    │ │
│  │ #tag  06-27 │ │ #tag  06-20 │ │ #tag 06-11 │ │  ← mono 元信息
│  └────────────┘ └────────────┘ └───────────┘ │

### AiEnglish 如何融入
- 这套与现有 AiEnglish 的深色聊天界面**最接近**，改造成本最低：只需把散装色换成统一中性灰 + 电光蓝令牌，气泡/输入/菜单立刻成体系，且与博客视觉一致。

## 方向 C · 暖光·杂志（Warm Editorial）

**一句话**：亲和消费风——暖陶土橙 + 鸭青 + 大圆角大图，柔和阴影，像 Ghost / Medium 获奖个人站，最「好接近」。

- **设计逻辑**：🎲 轮盘 / 大众亲和方向（友好杂志风）。
- **气质关键词**：温暖、友好、大图、圆润、有活力。
- **为什么适合**：受众更广、第一眼最讨喜；如果你希望博客「热闹好逛」、强调封面图与视觉吸引力，这套最合适。
- **注意**：圆角 + 柔阴影是消费级常见语言，需靠**克制配色 + 真实大图**避免落入模板感（卡片不堆 icon/tag/status 三件套）。

### 排版
- **标题**：Fraunces / 方正风雅宋 等「有个性的衬线」（暖、有表情）。
- **正文**：Noto Sans SC / 思源黑体，17–18px / 行高 1.75。
- 大圆角（`--radius-xl`）、柔和阴影（`--shadow-md`）、大封面图。

### 签名细节
- 封面图为主角，卡片大图 + 标题叠加；橙色作 CTA 与强调，鸭青作标签/链接。
- 微动效友好（卡片上浮 + 图片轻微放大）。
- 昼夜 Hero 图：白天暖阳大图，黑夜暖灯夜景，橙色贯穿。

## 三方向对比

| 维度 | A 子曰·墨 | B 终端·工坊 | C 暖光·杂志 |
|------|----------|------------|------------|
| 气质 | 文气克制 | 精确工程 | 温暖亲和 |
| 主色 | 朱砂红 | 电光蓝 | 陶土橙 + 鸭青 |
| 标题字 | 宋体 | 几何无衬线 | 个性衬线 |
| 信息密度 | 低（留白） | 高（网格） | 中（大图） |
| 第一眼吸引 | 内敛耐看 | 干净利落 | 最讨喜 |
| 与现有 AiEnglish 改造量 | 大 | **小** | 中-大 |
| 最契合的诉求 | 文学/原创内容 | 技术/工具属性 | 大众/视觉流量 |
| 反 slop 风险点 | 低 | 低（避开霓虹暗色） | 中（圆角模板感，靠真图破解） |

## 选型与下一步

请在复审时给出其一：

1. **选定一套**（A / B / C）。
2. **混搭**（例：「B 的网格与令牌精度 + A 的宋体标题与朱砂」「C 的大图卡片 + B 的中性克制」）。
3. **都不满意 / 想调整**：告诉我偏好（更暖/更冷、更密/更松、要不要中文衬线…），我重推。

> **我的倾向（仅供参考）**：若你最看重「子不语」的独特品牌与阅读质感，选 **A**；若最看重技术调性 + 最低改造成本（AiEnglish 几乎直接受益），选 **B**；若最看重大众吸引力与视觉热度，选 **C**。也很推荐 **B 的系统精度 + A 的宋体/朱砂签名** 这种混搭，兼得工程一致性与文气辨识度。

确认方向后：我会 ① 用 HTML 产出该方向 1–2 个关键页（首页 + 文章页）高保真原型；② 回填 [03-design-spec.md](./03-design-spec.md) 的待定色值/字体；③ 据此进入 [05-roadmap.md](./05-roadmap.md) 的代码优化。
